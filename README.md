1. Opis systemu

System Zarządzania Magazynem jest aplikacją webową stworzoną w technologii ASP.NET Core z bazą danych SQL i frontendem React.
Celem systemu jest umożliwienie firmom prowadzenia ewidencji:

dostawców,

odbiorców,

materiałów,

magazynów,

dokumentów magazynowych,

rezerwacji materiałów,

stanu magazynowego.
