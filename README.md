<h2><b>1. Opis systemu</b></h2>

<p>
Celem projektu jest stworzenie oraz wdrożenie systemu informatycznego wspierającego 
kluczowe procesy magazynowe, takie jak obsługa dostaw, wydawanie materiałów, 
prowadzenie rezerwacji oraz zarządzanie stanami magazynowymi. 
System został zaprojektowany jako aplikacja webowa, umożliwiająca pracownikom magazynu 
i administracji wygodne, szybkie i bezpieczne zarządzanie dokumentacją logistyczną 
oraz bieżące monitorowanie zasobów.
</p>

<h3><b>System zapewnia możliwość ewidencji:</b></h3>
<ul>
    <li>dostawców,</li>
    <li>odbiorców,</li>
    <li>materiałów,</li>
    <li>dokumentów magazynowych,</li>
    <li>aktualnych stanów magazynowych.</li>
</ul>

<h3><b>Obsługiwane procesy logistyczne:</b></h3>
<ul>
    <li>rejestrowanie i przyjmowanie dostaw,</li>
    <li>wydawanie materiałów,</li>
    <li>zatwierdzanie dokumentów,</li>
    <li>zarządzanie użytkownikami oraz ich uprawnieniami.</li>
</ul>

<h3><b>Role użytkowników w systemie:</b></h3>

<h4><b>Administrator</b></h4>
<ul>
    <li>posiada pełny dostęp do wszystkich modułów,</li>
    <li>zarządza użytkownikami i ich uprawnieniami,</li>
    <li>zatwierdza dokumenty i nadzoruje poprawność danych.</li>
</ul>

<h4><b>Magazynier</b></h4>
<ul>
    <li>tworzy i edytuje dokumenty magazynowe,</li>
    <li>dodaje dostawców i odbiorców,</li>
    <li>przegląda materiały oraz stany magazynowe,</li>
    <li>realizuje operacje przyjęć i wydań.</li>
</ul>

<p>
System stanowi nowoczesne narzędzie wspierające pracę magazynu, 
usprawniające obieg dokumentów i zapewniające przejrzystość operacji logistycznych.
</p>
